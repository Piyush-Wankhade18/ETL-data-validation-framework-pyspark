# ETL-data-validation-framework-pyspark


## Overview
This project is a reusable ETL validation framework built using PySpark for validating data pipelines in cloud and big data environments.

validations
- Row Count Validation
- Data Comparison (Full Load)
- Null Check
- Duplicate Check
- Aggregate Validation
- Logging & Reporting

Tech Stack
- PySpark
- Python
- Databricks
- SQL
